# Excalidraw Flowchart Library Context

For flowchart diagrams, use these conventions:

## Shapes
- **rectangle** – process/step/action
- **diamond** – decision/condition (branching)
- **ellipse** – start or end (terminator)
- **text** – standalone labels, titles, notes

## Flow
- Left-to-right or top-to-bottom
- Start (ellipse) → processes (rectangle) → decisions (diamond) → end (ellipse)
- Label decision arrows: "Yes", "No", "True", "False"

## Layout
- Align in rows or columns
- Use groupId for phases or branches
- No overlap; leave room for arrows
