# Network Shape Library (Draw.io)

Network topology and infrastructure diagrams.

## Layout
- Nodes: rectangles 120Ã—50, neutral colors (#e7f5ff, #f8f9fa)
- Spacing: 180px horizontal, 140px vertical between nodes. No overlap.
- Edges: arrows, orthogonal routing
- Group by: subnet, tier, or region using groupId

## Labels
- Router, Switch, Firewall, Server, Load Balancer
- Use short labels. For IP/hostname, add as secondary line or annotation
