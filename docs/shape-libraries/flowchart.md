# Flowchart Shape Library (Draw.io)

Standard flowchart shapes. Use rectangle, diamond, ellipse for process, decision, start/end.

## Shape mapping
- **Start/End**: ellipse, backgroundColor #d3f9d8, strokeColor #2f9e44
- **Process**: rectangle, rounded=1 style, backgroundColor #e7f5ff, strokeColor #1971c2
- **Decision**: diamond (rhombus), backgroundColor #fff3bf, strokeColor #f08c00
- **Document**: rectangle with document icon (use rectangle + label "Doc")
- **Arrow**: type "arrow", exitX/entryX for flow direction

## Flow
- Top-down or left-to-right. Be consistent.
- Spacing: 180px horizontal, 140px vertical between shapes. No overlap.
- Decision edges: label "yes" / "no" on arrows
- One start, one or more ends
